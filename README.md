# HairHistoryMemo

髪の履歴を記録して、美容師さんと共有するためのWebアプリです。

## はじめて触る人向け

1. バックエンド起動: `apps/main/README.md`
2. フロント起動: `apps/web` で `npm install` → `npm run dev`
3. ブラウザで `http://localhost:3000` を開く
